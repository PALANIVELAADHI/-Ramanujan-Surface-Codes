# RASF – Ramanujan Asymptotics for Surface-code Fault-tolerance

Exploratory project investigating whether modular forms (Jacobi theta functions + SL(2,ℤ) invariance) and Ramanujan's partition asymptotic can provide an analytical origin for the exponential logical error suppression P_L(d,p) ≈ exp(−κ(p) d) in surface codes below threshold.

**Status**: Proof-of-concept / theoretical exploration (2025–2026). 
Numerical results are illustrative and use strong approximations.

## Key claims (as explored)

- Logical error partition function Z_d(q) ≈ θ_3( i α d / π )
- Modular transformation → finite-size scaling + "modular gap" κ(p)
- Ramanujan asymptotic a_k(d) ≈ (1/(4k√3)) exp(π √(2k/3)) → upper bound on P_L via saddle-point like competition
